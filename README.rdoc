= hetzner-bootstrap

hetzner-bootstrap allows you to bootstrap a provisioned EQ Server from hetzner.de

Requirements

- get a webservice login (robots.your-server.de)
- the ip address of the shipped systems

== Installation

<tt> gem install hetzner-bootstrap</tt>
